Projetos

Monitoramento do Solo

Projeto integrado, focado na coleta, transmissão e visualização de dados ambientais em tempo real. O projeto emprega dispositivos Heltec ESP32 LoRa, sensores de temperatura e umidade do ar e do solo, além da utilização de tecnologias como MQTT e PostgreSQL hospedado na AWS.

Potabilidade da Água

Este projeto utiliza aprendizado de máquina para prever a potabilidade de amostras de água com base em parâmetros físicos e químicos. Através de um modelo de classificação treinado com dados históricos, a aplicação é capaz de determinar se uma amostra de água é potável ou não. O projeto foi desenvolvido utilizando Python, com técnicas de pré-processamento de dados para garantir a precisão das previsões.

Sensoriamento Espectral

Simulação para analisar a alocação espectral nas transmissões de TV no Brasil, focando nas faixas ociosas (TV White Spaces). Foram comparadas as técnicas cooperativa e não cooperativa de sensoriamento espectral. A simulação avaliou o desempenho do sistema em diferentes condições de interferência, como ruído AWGN, desvanecimentos por multipercurso e ruído impulsivo. O trabalho resultou em um artigo publicado internacionalmente na Austrália.

Conversor de Unidades

Este projeto consiste no desenvolvimento de um conversor de unidades com interface web, permitindo a conversão de diferentes unidades de medida de forma simples e intuitiva. A aplicação foi construída utilizando tecnologias como HTML, CSS, JavaScript e Python, proporcionando uma experiência interativa e eficiente.
